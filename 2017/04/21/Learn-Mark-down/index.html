关于Markdown

Markdown 是一种轻量级的「标记语言」。常用的标记符号也不超过十个，这种相对于更为复杂的HTML 标记语言来说，Markdown 可谓是十分轻量的。

详细请参考:markdown中文文档

2.mark down 列表

熟悉 HTML 的肯定知道有序列表与无序列表的区别，在 Markdown 下，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，
有序列表则直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格
